§ 00 · Principles

Adult-gamification · что мы НЕ делаем

Streak — side-info. Mastery — состояние, не награда. Calendar — данные, не панель почёта.

Banned · Duolingo-кладбище

  • 🔥 emoji для streak — pressure-design, infantil. Streak — это факт регулярности, а не подвиг.
  • +50 XP popups, leaderboards, leagues — вытесняют intrinsic motivation внешним цирком.
  • Confetti / fireworks / sparkle-bursts при mastered — каждый visit = "отпраздновали третий раз". Адские.
  • Trophy / badge / star / medal — иконография цирка.
  • Daily quest, login bonus, motivational quotes — корпоративный токсичный позитив.
  • Re-celebration: glow на каждое открытие mastered ring. Один acknowledged glow → static.

Approved · Adult-tone matrix

  • Streak = inline dot-row рядом с именем. Числу 5 веришь, не отмечаешь.
  • Mastery = ring 24–32px фоном. Lime ✓ при mastered. Один 600ms glow на pattern за всю историю.
  • Calendar = GitHub-graph. Opacity по study-load (theory=3 vs drill=1) — не сводка минут.
  • Last session = single-line recap (вчера: 2h, +12% mastery on Past Simple negative). Connects yesterday→today.
  • Copy: «К повтору» (не «Просрочено»), «Зоны фокуса» (не «Слабые места»), «В автоматизме» (не «Automated»).
  • Glow rule: mastered transition + один miss-recovery → дальше static. mastery_acknowledged boolean per pattern.
