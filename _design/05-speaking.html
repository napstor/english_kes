§ 00 · System

Что строим и как

Speaking-шаг — финальный gate (§3.5 + §6 + §7.3/7.4 брифа). 6 состояний, split-layout, no MVP three-cards anti-pattern.

Layout = split, не stack. Recording-console (~600px) слева, pronunciation/intonation (~640px) справа. До scoring правая колонка — empty-state с подсказкой "Запиши, чтобы получить разбор". В MVP-mock из вдохновений три отдельных карточки Yours / Slow / Normal были как big cards — у нас они PlaybackRow, inline.
Waveform = CSS breathing. Не real-time analyzer для MVP — 24 столбца с staggered animation-delay и keyframes, это считывается как «идёт запись» при 60fps без AudioContext. Точная реализация на v2.
Recognition = TokenChip stream. Каждое слово — pill: matched lime · mismatched red strike · missing dashed ghost · extra amber. Глазом пробегается за 2 секунды.
IntonationGuide = arrows под словами. ↑ rising / ↓ falling / • stress / / pause из §7.4 брифа — глифы под жирными словами, не отдельным rich-blob. Mono font для стрелок, sans для слов.

§ 05b · IntonationGuide · mobile · 375

Как фраза со стрелками умещается в узкий вьюпорт

Phrase читается как обычный текст и переносится по словам. Стрелки/глифы прибиты к слову через display:inline-flex; flex-direction:column, поэтому при line-break каждая строка несёт свой spline-fragment. Никаких отдельных SVG-полилиний которые могут поехать.

IG-M3 · long · scroll fallback
Интонация · 11 слов
If↑ I had• known about it• I would have• come↓
swipe
IG-M3 · >8 слов · horizontal scroll + fade-out + visual hint

Решение:  1) до 6 слов → одна строка, spline целиком;  2) 6–8 → flex-wrap, каждая строка несёт свои стрелки (нативно благодаря inline-flex column);  3) >8 → flex-wrap:nowrap + overflow-x:auto + правый fade-mask + микро-arrow hint "swipe". Это лучше чем переносить на 3 строки и терять читаемость intonation-контура.

§ 08 · Score gate · sober escape

Что происходит, когда не попал в 75

Три ветки. Первые две attempts — обычный retry. Третья — открывается escape. Жёсткий пол на 60 — там retry без альтернатив.

Логика без флагов:  score ≥ 75 → primary "Дальше" lime ·  60–74 attempt 1–2 → primary "Записать снова" + ghost "Дальше" disabled ·  60–74 attempt 3+ → "Continue anyway · NN" появляется как ghost, переход помечает phrase в weak-spots для review ·  <60 → escape отсутствует, retry единственная опция (но Skip всей записи доступен через actions bar).
