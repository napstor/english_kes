§ 03 · Empty states

Compact glass · 1-line · 1 CTA

Не «You haven't done anything yet, click here to get started!». Спокойная констатация + один следующий шаг.

К повтору сегодня — пусто Все паттерны на нужной частоте. Загляни завтра — будет 6 новых для review.
Ничего не нашлось По «past perfect continuous» в твоей программе нет шагов. Проверь написание или переключись на EN.
Первый день в курсе Recap появится здесь, как только закончишь первое занятие. Это будет завтра — старт с урока 1.
Принцип: empty state — НЕ рекламная пауза. Без иллюстраций «коробок», маскотов, восклицательных знаков. Glyph 56px (lucide thin), 1 короткий заголовок (≤ 6 слов), 1 строка контекста, 1 CTA. Если CTA очевиден из контекста — можно без CTA вообще.
§ 04 · Error states · Offline

Inline Callout · 3 severities · top-banner offline

Никаких modal-popup'ов «Что-то пошло не так». Inline там, где случилась ошибка. Top-banner — для глобального state.

Validation · inline
Email
Похоже, в email опечатка — нет домена
Validation: красная рамка + иконка + 1 строка контекста. Не блокирует submit (он сам не пройдёт). Цель — помочь увидеть, не отчитать.
Network · inline retry
Не удалось отправить запись
Сервер не ответил за 8 секунд. Запись сохранена локально — попробуй ещё раз.
Network/timeout: amber, не red — это transient. Запись/ввод сохранены локально (mention!). Retry + Skip как опции.
5xx · server
Сервер недоступен
Это не у тебя — что-то у нас. Прогресс сохранён локально. Мы уже знаем и чиним.
Tracea8f3.b2.20251014
5xx: red. «Это не у тебя» — снимает с пользователя самообвинение. Trace ID для support. Link на status page.
OfflineПрогресс сохраняется локально. Мы синхронизируем, как только сеть вернётся — ничего не потеряется.
Offline banner · top-fixed · auto-dismiss когда сеть восстанавливается. Показывается через navigator.onLine + ping. Никогда не блокирует UI (overlay/modal) — все основные flows работают офлайн с локальной очередью.
§ 05 · Speaking · 10s+ recognizing fallback

Прогрессивный copy при долгом распознавании

Дополнение к §04 Stage 5. После 10 секунд в processing — даём escape, не молчим.

2s · 0–2s
Распознаём…
Спокойный feedback. Spinner + 1 слово. Никакой паники.
5s · 2–10s
Чуть дольше обычного Сервер задумался. Сейчас догрузим — оставайся на связи.
Acknowledgement → меняем язык. «Чуть дольше обычного» — честно, без апологии.
10s+ · escape
Не удалось распознать Связь подвела. Запись сохранена — попробуй ещё раз.
10s — порог терпения. Даём escape: retry или skip. Запись не теряется (важно — это minutes of work для пользователя).
Implementation: в RecordingConsole добавить таймер от setStartProcessing(now) до scoring. На 2s/10s переходов меняем recognizingPhase: 'short' | 'medium' | 'long'. Long-state открывает escape-buttons. Запись (Blob) держим в memory до next attempt — retry не требует повторной записи.
§ 06 · A11y audit

WCAG AA · keyboard · screen reader · prefers-reduced-motion

Brief §12: «WCAG AA по всем text-on-surface, focus-ring видимый, screen reader осмысленный». Чек-лист для каждого экрана + global rules.

Global rules · check before merge
Focus-ring 2px --accent-primary + 4px halo на ВСЕХ интерактивных
.btn, .icon-btn, .nav-item, input, [role="button"], a · :focus-visible only
A.01
Контраст fg/bg ≥ 4.5:1 для body, ≥ 3:1 для large display
--fg-primary on --bg-base · checked. --fg-muted on --bg-elevated · 5.2:1 · ok
A.02
Hit target ≥ 44×44px на touch (mobile), 32×32px на desktop
.icon-btn 36×36 · ok desktop · mobile-обёртка с padding до 44
A.03
Tab order логичный · top→bottom, left→right
TopBar → Sidebar → Main → Footer-actions. Skip-link на main · h-tab-1
A.04
prefers-reduced-motion отключает декор (waveform, shimmer, glow)
@media (prefers-reduced-motion:reduce) · animation:none на .skel, .waveform, .mr.glow
A.05
Все иконки — либо aria-hidden="true", либо парный aria-label
.icon-btn[aria-label="Settings"] · svg внутри aria-hidden. Никогда не оба
A.06
Color-only meaning запрещён · pron-score «red» дублирован числом 45 + текстом «Заметные ошибки»
Red dot ≠ semantic — рядом текст всегда
A.07
Screen reader: heading hierarchy h1→h2→h3 без skip · landmark роли
main, nav, header, aside (StepRail) · в каждом screen ровно один h1
A.08
Esc закрывает все overlay (AdminSheet, TutorChat, CourseDrawer, focus-mode)
useEscapeKey hook · возврат focus на triggering element
A.09
Live regions: ScoreBadge, recognizing-phase, offline-banner
aria-live="polite" для score · "assertive" только для error/offline
